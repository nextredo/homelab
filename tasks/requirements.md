# Stuff
## Qualities
### Required
- [ ] Is FLOSS
- [ ] Is one of:
  - Maintained
  - Feature-complete, stable, maintenance mode
- [ ] Simple
- [ ] Robust
- [ ] Has a lot of stars on GitHub
  - Less likely to become unmaintained
  - Likely better bugfixes, support
  - Likely better docs, code

### Desired
- [ ] N/A

## Functionality
### Required
- [ ] Ability to limit tasks in progress
  - (tasks in a list)
  - Stops doing too many things at once
    - Getting sidetracked
    - Feeling overwhelmed / burnt out
    - Decision paralysis
- [ ] Kanban style
  - This style really appeals to me
  - Simplistic, functional, good
- [ ] Customisable columns/lists
  - Can change what lists I have in a board
  - Have more than 3 boards
  - Other labels from [here](task-lifecycle)
- [ ] Tags & filtering
  - Allows further categorisation
- [ ] Projects & sub-projects
  - To represent different areas of my life
  - Computer shit (software, linux, projects)
  - Self-improvement stuff (podcasts, habits)
  - German
  - Gym
- [ ] **Interoperability**
  - Task & database formats can easily be moved between different kanban apps
  - No vendor lock-in

### Desired
- [ ] Timers for tasks
  - Unsure if I would use this often
- [ ] Task dependencies
  - Unsure if I would use this often
  - Weary of the task management system becoming a task itself, instead of a tool
- [ ] Recurring tasks
- [ ] General functionality
  - Move tasks between boards
- [ ] Written in languages I want to learn
- [ ] Has CopyLeft licensing (GPL)
  - Harder to be rugpulled into a proprietary / paid solution
- [ ] A mobile app
  - But a functional mobile web interface is ok too
- [ ] Date logging
- [ ] Notes app tie-in
